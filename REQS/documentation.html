BitlySDK

The Bitly Interview SDK offers a few methods that would help you in creating and querying a Bitlink (Bitly-branded short URL)

BitlySDK
Parameters
options (Object)
Name Description
options.login string Username of the Bitly account
options.apiKey string API Key of the Bitly account
Example
// Usage:
const bitlySDK = new BitlySDK({
  login: 'hunter2',
  apiKey: 'R_d41d8cd98f00b204e9800998ecf8427e'
});
Instance Members
▸ shorten(longUrl)

Given a long URL, creates a Bitlink, returns a JSON containing the Bitlink short URL.

shorten(longUrl: string): Promise
Parameters
longUrl (string) The long URL to be shortened
Returns
Promise: Promise containing the JSON: { long_url: string; url: string; }
Throws
  • TypeError: Error produced by the browser (e.g. Timeout, XSS error)
  • Error: HTTP/API Errors (e.g. "500 INVALID_APIKEY", "500 MISSING_ARG")
Example
// Usage:
bitlySDK.shorten('http://google.com/').then(function(result){
  console.log(result);
});

// Response JSON:
{
  long_url: "http://google.com/",
  url: "http://bit.ly/ze6poY",
  ...
}
▸ expand(shortUrl)

Given a Bitlink, returns a JSON containing the original long URL.

expand(shortUrl: string): Promise
Parameters
shortUrl (string) The short URL to be expanded
Returns
Promise: Promise containing the JSON: { long_url: string; short_url: string; }
Throws
  • TypeError: Error produced by the browser (e.g. Timeout, XSS error)
  • Error: HTTP/API Errors (e.g. "500 INVALID_APIKEY", "500 MISSING_ARG")
  • Error: NOT_FOUND - Bitlink was not found
Example
// Usage:
bitlySDK.expand('http://bit.ly/ze6poY').then(function(result){
  console.log(result);
});

// Response JSON:
{
  short_url: "http://bit.ly/ze6poY",
  long_url: "http://google.com/",
  ...
}

▸ clicks(shortUrl)

Given a list of Bitlinks, returns a JSON containing the click count those pages.

clicks(shortUrl: Array<string>): Promise
Parameters
shortUrl (Array<string>) The short URL to be queried
Returns
Promise: Promise containing the JSON: Array<{ short_url: string; global_hash: number; }>
Throws
  • TypeError: Error produced by the browser (e.g. Timeout, XSS error)
  • Error: HTTP/API Errors (e.g. "500 INVALID_APIKEY", "500 MISSING_ARG")
Example
// Usage:
bitlySDK.clicks(['http://bit.ly/ze6poY', 'http://bit.ly/a']).then(function(result){
  console.log(result);
});

// Response JSON:
[{
  short_url: "http://bit.ly/ze6poY",
  global_clicks: 442,
  ...
}, {
  short_url: "http://bit.ly/a",
  global_clicks: 431242,
  ...
}]
